# Business Sales Analytics

## Overview

This project analyzes retail sales data to identify business trends, product performance, regional sales distribution, and profitability. The analysis was performed using Python and focuses on transforming raw business data into meaningful insights that support data-driven decision-making.

## Objectives

* Analyze sales trends over time
* Identify top-selling products
* Evaluate category-wise profitability
* Compare regional sales performance
* Generate business insights and recommendations

## Dataset

**Dataset Used:** Superstore Sales Dataset

The dataset contains information related to:

* Orders
* Products
* Sales
* Profit
* Categories
* Regions
* Customers

## Data Cleaning

The following preprocessing steps were performed:

* Removed duplicate records
* Checked for missing values
* Converted date columns to datetime format
* Prepared data for analysis and visualization

## Analysis Performed

### 1. Monthly Sales Trend Analysis

Analyzed sales performance over time to identify patterns and trends.

### 2. Top Selling Products

Identified the products generating the highest revenue.

### 3. Regional Sales Analysis

Compared sales performance across different regions.

### 4. Category Profit Analysis

Evaluated profitability across product categories.

## Key Insights

* Sales performance varies across different periods.
* A small group of products contributes significantly to overall revenue.
* Regional performance differs across markets.
* Product categories show varying levels of profitability.

## Recommendations

* Focus on high-performing products and categories.
* Strengthen strategies in underperforming regions.
* Use sales trends for inventory and demand planning.
* Monitor profitability alongside revenue growth.

## Conclusion

This project demonstrates how Python can be used for business sales analytics through data cleaning, visualization, and insight generation. The findings can help businesses make informed decisions and improve overall performance.
